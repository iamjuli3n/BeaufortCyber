Getting Started

Basic Installation

Steps for getting started with the template:

  1. Open .../Template Folder to find all the Templates Files
  2. Before making any changes, ensure you have a local development environment set up. You can use tools like Live Server or open the HTML file in your preferred web browser.
  3. Here is the file structure of the template:
    • index.html - Main Html File
    • index2.html - Home-2 Html File
    • assets/css - Stylesheets Folder
    • assets/js - Javacripts Folder
    • assets/img - Images Folder
    • assets/webfonts - Fontawesome Folder
    • assets/flaticon - Flaticon Folder
    Other files can be used according to your convenience,.
  4. You can now replace content/images with your own and generate your beautiful landing page.
  5. Now that you have followed these steps your basic installation is complete and ready to flaunt your site to the world.
